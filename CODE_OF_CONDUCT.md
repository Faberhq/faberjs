# Community Standards

FaberJS is an open source project. We want a productive and welcoming environment for all contributors, regardless of experience level.

## Expected Behavior

- Communicate respectfully and constructively
- Welcome and support contributors of all backgrounds and skill levels
- Give and receive feedback on code and ideas, not people
- Stay on-topic in issues, PRs, and discussions

## Unacceptable Behavior

- Harassment, insults, or personal attacks of any kind
- Discriminatory language or conduct
- Sharing others' private information without consent
- Conduct that would be considered unprofessional in any working environment

## Reporting

Report any concerns to the maintainers at **conduct@faberjs.dev**. All reports are reviewed confidentially.

## Scope

These standards apply to all project spaces: GitHub issues, pull requests, discussions, and any other FaberJS community channels.
